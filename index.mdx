## Get Started in Minutes

<Steps>
  <Step title="Create Your Account">
    Sign up for a Streamline account and get instant access to our sandbox environment.

    [Sign Up Now →](https://dashboard.streamline.example.com/signup)
  </Step>

  <Step title="Get Your API Keys">
    Generate your API credentials from the dashboard—separate keys for sandbox and production.
```bash
    # Your API credentials
    Sandbox: sk_sandbox_...
    Production: sk_live_...
```
  </Step>

  <Step title="Make Your First Request">
    Create a stored account and process your first payment.

    <CodeGroup>
```bash cURL
    curl https://sandbox.api.streamline.example.com/v3/stored-accounts \
      -H "Authorization: Bearer sk_sandbox_..." \
      -H "Idempotency-Key: $(uuidgen)" \
      -H "Content-Type: application/json" \
      -d '{
        "type": "ach",
        "location_id": "loc_123",
        "account_number": "123456789",
        "routing_number": "021000021",
        "account_type": "checking",
        "account_holder_name": "Jane Doe"
      }'
```
```python Python
    import requests
    import uuid

    response = requests.post(
        'https://sandbox.api.streamline.example.com/v3/stored-accounts',
        headers={
            'Authorization': 'Bearer sk_sandbox_...',
            'Idempotency-Key': str(uuid.uuid4()),
            'Content-Type': 'application/json'
        },
        json={
            'type': 'ach',
            'location_id': 'loc_123',
            'account_number': '123456789',
            'routing_number': '021000021',
            'account_type': 'checking',
            'account_holder_name': 'Jane Doe'
        }
    )

    account = response.json()
    print(f"Account created: {account['id']}")
```
```javascript Node.js
    const axios = require('axios');
    const { v4: uuidv4 } = require('uuid');

    const response = await axios.post(
      'https://sandbox.api.streamline.example.com/v3/stored-accounts',
      {
        type: 'ach',
        location_id: 'loc_123',
        account_number: '123456789',
        routing_number: '021000021',
        account_type: 'checking',
        account_holder_name: 'Jane Doe'
      },
      {
        headers: {
          'Authorization': 'Bearer sk_sandbox_...',
          'Idempotency-Key': uuidv4(),
          'Content-Type': 'application/json'
        }
      }
    );

    console.log('Account created:', response.data.id);
```
    </CodeGroup>
  </Step>

  <Step title="Process a Payment">
    Create a payment against your stored account.
```bash
    curl https://sandbox.api.streamline.example.com/v3/payments \
      -H "Authorization: Bearer sk_sandbox_..." \
      -H "Idempotency-Key: $(uuidgen)" \
      -H "Content-Type: application/json" \
      -d '{
        "location_id": "loc_123",
        "payment_method": "ach",
        "stored_account_id": "acct_...",
        "amount": 100.00,
        "direction": "debit",
        "description": "Test payment"
      }'
```
  </Step>

  <Step title="Set Up Webhooks">
    Get real-time notifications about payment events.
```bash
    curl https://sandbox.api.streamline.example.com/v3/webhook-endpoints \
      -H "Authorization: Bearer sk_sandbox_..." \
      -H "Content-Type: application/json" \
      -d '{
        "url": "https://your-app.com/webhooks",
        "events": ["payment.created", "payment.settled", "payment.failed"]
      }'
```
  </Step>
</Steps>
