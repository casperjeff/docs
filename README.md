# Payments Platform Docs (Mintlify)

This repository contains a Mintlify documentation site scaffolded for a payments platform. All section and subsection pages are created as placeholders.

## Prerequisites
- Node.js 18+
- npm (or yarn/pnpm)

## Run locally
```bash
npm install
npx mintlify dev
```
If you prefer using a config file, `mint.config.mjs` is also included.

## Deploy
Push this repo to GitHub and connect it to Mintlify. The API reference is generated from `api/openapi.yaml`.
